COLOR

Bootstrap comes equipped with colors built in to the css. Each one is designed to be a placeholder for your own color of choice down the road.



You can assign color to background and text elements through the use of bootstraps built in css. In this example we are targeting the background color using the .bg property. We would use the .text property to specify a text color

class = ".bg-primary"
class = ".bg-secondary"
class = ".bg-success"
class = ".bg-danger"
class = ".bg-warning"
class = ".bg-info"
class = ".bg-light"
class = ".bg-dark"
class = ".bg-white"
