THE GRID SYSTEM

Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content on a page. It's built using flexbox to allow for fully responsive capabilities!

12

The grid is made up of sections of 12. Always remember your classes should add up to 12 or less, NEVER OVER 12! You start using grid by calling a container to place all the content inside of. Next you will call a row to place a horizontal axis in the container. Then comes the most important part, calling your column class(s). Your column classes will divide your row up into vertical sections based on how many you specify. To get a specific sized column, you can add a breakpoint along with a numeric value up to 12. If you do not specify a size, bootstrap will automatically set a size based on the content inside. A quick example would look something like col-sm-12. What this is saying is at a small screen size, I want the column to stretch the whole width of the container (remember the rule of 12).

Lets break this down. Here we have 1 container which is holding everything together. Within that that container we have 6 row classes. Each row contains its specified column breakpoint and size. To make it easy we set each breakpoint to small, this means that once the screen resizes below a small, the content will do what bootstrap feels is best practice. Usually this results in items stacking which is what happens with our example here. Resize it and see for yourself!
