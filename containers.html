THE GRID SYSTEM

Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content on a page. It's built using flexbox to allow for fully responsive capabilities!

12

The grid is made up of sections of 12. Always remember your classes should add up to 12 or less, NEVER OVER 12! You start using grid by calling a Container to place all the content inside of. Next you will call a Row to place a horizontal axis in the container. Then comes the most important part, calling your Column class(s)
